# 🇿🇦 South African Entry-Level Technology Jobs Analysis

## Project Overview

This project analyzes entry-level technology job opportunities in South Africa. The objective was to identify the most in-demand technical skills, programming languages, software tools, and work requirements by analyzing real job postings.

The project demonstrates practical data analysis skills including data collection, data cleaning, exploratory data analysis (EDA), and visualization using Python.

---

##Objectives

- Collect entry-level technology job postings
- Clean and prepare the dataset
- Explore hiring trends
- Identify the most requested technical skills
- Visualize findings using Python

---

## Dataset

The project includes:

- Raw job postings dataset
- Cleaned dataset used for analysis

Files included:

- `AGN week 2 raw job posts - Sheet1.csv`
- `AGN_week2_cleaned.csv`

## Key Insights

The analysis identifies:

- Most common entry-level IT roles
- Popular programming languages
- Frequently requested technical skills
- Common software tools
- Hiring trends within South Africa
